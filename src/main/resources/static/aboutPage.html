What is Personality Analyzer ?

This project was developed for predicting personality of users based on their Twitter posts. Personality analysis was done for each individual. There is a tool for gathering tweets from Twitter platform and also one more tool for analyzing Turkish words of tweets. The tweets are only in Turkish, in another language will not be analyzed in the project. Tweets also include images as well as text. But in this project scope, only texts are processed and word content of tweets are analyzed. Words are grouped according to personality prediction models. The main personality model is used in the project is the Big Five Personality Model. There is a simple user interface that contains operations as a minimum. Most of the works are done as a background / batch process. In the background, gathering tweets and grouping words in tweets and analyzing them is done. Users are able to only see the results of personality predictions. The results of this project can be used in any sector. For example, human resources can use the results of this project to estimate the personality of workers that will be hired in companies. The project can be used in education field to predict teachersâ€™ behaviors towards students. Another domain can be social media applications such as music application can use the result of this project for a new content recommendation for a user.
